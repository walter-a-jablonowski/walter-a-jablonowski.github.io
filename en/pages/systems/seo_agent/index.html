What is it?

SEO AI Agent is an AI agent that keeps your website fit for search on its own – for Google as much as for AI answer engines (GEO). It runs on a schedule, collects real data about your site, finds weak spots and proposes targeted improvements. Nothing goes live unasked: the agent puts its changes on a dedicated branch for you to review and approve.

  • Optimizes for classic search and AI search (GEO) in one pass
  • Works with real data: Google Search Console, GA4, SERP, AI citations
  • Targets meta, content, JSON-LD and alt texts – and writes new pages (DE+EN)
  • Changes land on a branch for approval – the agent never pushes

Features

Audit on real data

Crawls the site and reads Search Console, GA4, SERP and AI citations.

Targeted edits

Changes meta titles, text blocks, JSON-LD and alt texts – precisely, not wholesale.

Visible in AI search

GEO: checks whether ChatGPT, Gemini & Perplexity cite you – and works on it.

New pages

For keywords with no ranking page it writes whole pages (DE+EN) and links them.

Approval by branch

Each run commits to a branch for review – you merge what goes live.

How it works

  1. The agent collects data and measures where the site stands
  2. It researches gaps and builds a briefing for the AI model
  3. The model returns targeted edits – and new pages for missing keywords
  4. Flow: Collect → Measure → Research → Brief → Edit → Guardrails → Commit

Before the commit, guardrails check the entire edited site – broken links, JSON-LD, meta lengths, duplicate titles, valid HTML. If a check fails, every file is restored and nothing is committed.

The language model is interchangeable: cloud (OpenRouter with Gemini 3.5 Flash, Google Gemini) or local (Ollama) – depending on how much should stay in-house.

Tailor-made for you

SEO AI Agent is not a rigid off-the-shelf tool but modular by design, tuned to your website and your goals. Every building block is interchangeable:

  • Target keywords, edit budget and schedule freely chosen per website
  • Data sources toggled on: Search Console, GA4, SERP, AI citations
  • AI model of your choice: cloud for speed or local (Ollama) for maximum confidentiality
  • Design-sensitive pages spared: only small edits (meta, alt, JSON-LD), no new sections
  • Run either as a proposal (propose) or as a commit for approval

Control & safety: The agent is sandboxed to your website directory – it never writes outside it. Every file is backed up before a change; if a guardrail fails, everything is restored. Changes land on a branch for approval – the agent never pushes.

Frequently asked questions

What does it cost?

Setup at a fixed price · plus ongoing usage costs (AI model/APIs). The price depends on scope and data sources – the initial consultation is free.

Does the agent change my site without oversight?

No. Each run commits to a dedicated branch; you review the diff and merge only what should go live. The agent never pushes. In „propose“ mode it only writes suggestions.

What is GEO / AI search?

GEO stands for „Generative Engine Optimization“ – visibility in AI answers (ChatGPT, Gemini, Perplexity). The agent checks whether they cite you and works on it deliberately.

Does it work with my website?

The agent is built for websites in a Git repository (e.g. GitHub Pages) with a German and English language version. I’m happy to check other setups in an initial call.

Where does my data run?

The agent runs on your machine or server and is sandboxed to your website directory. With a local model (Ollama), your content does not leave your premises.

Which AI is behind it?

By default OpenRouter with Gemini 3.5 Flash; Google Gemini or a local model (Ollama) are equally possible. The model can be swapped without rebuilding the agent.
