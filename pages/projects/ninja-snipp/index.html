NinjaSnipp

Ein Snippet-Manager, schnell und kosteneffizient mit KI erstellt

  • Aufwand: mit KI nur ein paar Nachmittage als Nebenprojekt
  • Manueller Entwicklungsaufwand wäre: locker 2-3 Monate (viel Code, Beispiel: siehe controller.js)

Stand: Alle Grundfunktionen implementiert, Standards wie Login und erweiterte Funktionen folgen noch

Projekt ansehen: GitHub

Entwicklungsprozess

Verwendete Tools:

  • Windsurf IDE
  • Claude Sonnet, GPT-5, Grok Code Fast 1

Prinzip

Als erfahrener Entwickler steigere ich meine Effizienz durch den Einsatz von KI. Indem ich möglichst viel Entwicklungsarbeit an eine KI delegiere, kann ich das Verhältnis von Wert zu Zeit verbessern. Arbeitsablauf:

Initialer Prompt

Die Entwicklung begann mit einem initialen Prompt, der die wichtigsten Informationen enthält. Die KI erstellte eine erste Version der App.

Feinabstimmung

Eine Reihe von Prompts diente dazu, Detailfunktionen der App zu korrigieren/fertigzustellen. Der Code wurde vollständig von der KI geschrieben, unter menschlicher Aufsicht.
